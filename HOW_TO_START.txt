================================================================================
  🚀 HOW TO DEPLOY TO RENDER - SIMPLE GUIDE
================================================================================

✅ CODE ALREADY PUSHED TO GITHUB!
   Repository: https://github.com/AkashMaurya/onlinezoommeeting

================================================================================
  STEP 1: GO TO RENDER
================================================================================

1. Open browser and go to: https://render.com
2. Click "Get Started for Free" or "Sign In"
3. Sign up with GitHub (recommended) or email
4. Verify your email if needed

================================================================================
  STEP 2: CREATE WEB SERVICE
================================================================================

1. Click "New +" button (top right corner)
2. Select "Web Service"
3. Click "Connect account" if not connected to GitHub
4. Find your repository: "onlinezoommeeting"
5. Click "Connect" button next to it

================================================================================
  STEP 3: FILL IN THE FORM (IMPORTANT!)
================================================================================

Copy these values EXACTLY:

┌─────────────────────────────────────────────────────────────────┐
│ Name:                                                           │
│   online-church-meeting                                         │
│                                                                 │
│ Region:                                                         │
│   Choose closest to you (e.g., Oregon, Ohio, Frankfurt)        │
│                                                                 │
│ Branch:                                                         │
│   main                                                          │
│                                                                 │
│ Root Directory:                                                 │
│   (leave empty)                                                 │
│                                                                 │
│ Runtime:                                                        │
│   Python 3                                                      │
│                                                                 │
│ Build Command:                                                  │
│   pip install -r requirements.txt                               │
│                                                                 │
│ ⚠️ Start Command (REQUIRED - COPY EXACTLY):                     │
│   uvicorn main:app --host 0.0.0.0 --port $PORT                  │
│                                                                 │
│ Plan:                                                           │
│   Free                                                          │
└─────────────────────────────────────────────────────────────────┘

================================================================================
  STEP 4: DEPLOY
================================================================================

1. Scroll to bottom
2. Click "Create Web Service" button
3. Wait 5-10 minutes (you'll see build logs)
4. When you see "Live" badge - it's ready!
5. Click the URL at top to open your app

================================================================================
  YOUR APP URL WILL BE:
================================================================================

https://online-church-meeting.onrender.com

(or whatever name you chose in Step 3)

================================================================================
  TEST YOUR APP
================================================================================

1. Open the Render URL
2. Enter your name
3. Click "Create New Meeting"
4. Allow camera/microphone
5. Copy meeting link
6. Open in another tab/device
7. Join the same meeting
8. Test chat, reactions, video!

================================================================================
  ⚠️ MOST COMMON ERROR
================================================================================

ERROR: "Start command is required"

SOLUTION: Make sure you entered this in "Start Command" field:

  uvicorn main:app --host 0.0.0.0 --port $PORT

Copy it EXACTLY as shown above!
